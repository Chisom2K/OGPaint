Version 0.3.0 (9/22/23)
New Features:
	- New drawing tools!
		- Freehand/"Pencil" - go crazy and draw everywhere
		- Ellipse (with possible fill)
		- Rectangles (with possible fill)
			- Width on all shapes changes when you change 'Line Width'
		- Square (with possible fill)
		- Circle (with possible fill)
		-Triangle 
		- Text in any color you desire
		
	- Random Color!
		- Don't know what color you want? Hit 'Random Color' until one
		  catches your eye!
	- Keyboard commands
		- Alt+F/Alt+O/Alt+H opens all of the menu bar options
		- Alt+S and Ctrl+S saves!
		- Ctrl+[ zooms out and Ctrl+] zooms in
	- New menu bar option
		-Zoom in and zoom out now available!
	- Text label
		- Now you can see what color you're using
Features in-progress:
	-To place a an Icon in a label 
	- Add a copy and Paste
		
Bugs:
	-Triangle is a bit weirdly shaped(surely to be fixed soon)

Version 0.2.0 (9/16/23)
New Features:
	- About and Help
		- If you're confused, those are the best places to go for info
	- Draw line
		- Drag and release to draw a line from point A to point B
	- Resize image
		- basically just scroll bars
	- Tab
		- Currently only one can be opened but in the future more tabs can be
		  added
	- Paint Tools
		- Sits right under the Menu Bar; all of the tools you'll need to paint 
		  things
Features in-progress:
	- Random color
		- Gives the user a random color at the click of a button
	- More Tabs
	- "Don't show me the release notes again" checkbox
Bugs:
	- Closing the tab is not good and requires a reboot
